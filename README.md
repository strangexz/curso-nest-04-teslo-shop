# Teslo API

1. Clonar el proyecto
2. Ejecutar `npm install`
3. Clonar el archivo **_.env.template_** y renombrarlo a **_.env_**
4. Cambiar las variables de entorno
5. Levantar la base de datos

   ```bash
   docker-compose up -d
   ```

6. Ejecutar: `npm run start:dev`
